# Test Writer Agent

## Overview
AI agent that analyzes source code and generates comprehensive unit tests for uncovered functions and edge cases.

## Tech Stack
- TypeScript, Node.js

## Commands
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Features
- Configurable AI model backend
- Structured output formatting
- Error handling and retries
- Rate limiting
- Verbose logging mode
- JSON and markdown output
- Git integration
- CI/CD compatible
- Dry-run mode
- Progress indicators

## Design Notes
- Premium, modern CLI with dark mode support
- Smooth animations and micro-interactions
- Accessible (WCAG 2.1 AA compliant)
- No external API keys required for core functionality

## Testing
- Unit tests for core logic
- Target ≥80% code coverage
