# Code Review Agent

Automated code review with style and bug suggestions.

## Features
- File/diff input
- Style checking
- Bug detection
- Security scan
- Performance tips
- Complexity rating
- AI-powered improvement suggestions
- Generate detailed analysis report

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
