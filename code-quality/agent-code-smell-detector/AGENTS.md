# Code Smell Detector Agent

## Overview
AI agent that scans codebases for common code smells and suggests refactoring patterns

## Tech Stack
- **Stack:** Python, LangChain
- **Build:** `pip install -r requirements.txt` then `python main.py`

## Features
- Detect common code smells
- Long method detection
- God class identification
- Feature envy analysis
- Refactoring suggestions
- Severity scoring
- File-by-file report
- CI integration support

## File Structure
- `main.py — agent entry`
- `agent/ — agent modules`
- `requirements.txt — dependencies`
- `README.md — documentation`

## Design Guidelines
- **Theme:** Dark mode with Ocean Blue palette
- **Primary:** `#0984E3` | **Secondary:** `#74B9FF`
- **Accent:** `#FDCB6E` | **Background:** `#0A141A`
- **Border Radius:** 12px | **Font:** Inter or system font stack
- **Animations:** Smooth transitions (200-300ms ease)
- **Style:** Modern, premium, glassmorphism accents

## Requirements
- Fully functional — no placeholder content
- Responsive design | Accessible | No external API keys
