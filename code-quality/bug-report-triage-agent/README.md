# Bug Report Triage Agent

Auto-triage bug reports by severity.

## Features
- Bug report input
- Severity classification
- Component detection
- Duplicate check
- Priority scoring
- Assignee suggest
- Auto-assign labels and tags
- Triage report

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
