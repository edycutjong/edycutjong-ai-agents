# AGENTS.md — Regex Tester

## Overview
Extract matches using a common pattern name.

## Tech
- Python 3.10+
- python-dotenv
- pytest

## Features
- Cmd Test
- Cmd Validate
- Cmd Explain
- Cmd Library
- Cmd Extract

## File Structure
- `README.md`: Readme.Md
- `agent/`: Module directory
- `config.py`: Configuration
- `main.py`: Entry point
- `requirements.txt`: Dependencies
- `tests/`: Module directory

## Commands
- `pip install -r requirements.txt` — Install deps
- `python main.py` — Run agent
- `pytest tests/` — Run tests
