## Overview
AI agent to draft, schedule, and engage with social content based on brand voice.

## Tech Stack

- **AI:** OpenAI API / Anthropic API / Gemini
- **Framework:** LangChain or custom loop
- **Interface:** CLI or Streamlit

- `config.py`: Settings
