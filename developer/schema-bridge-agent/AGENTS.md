# Schema Bridge Agent

## Overview
Agent that maps fields between two legacy databases.
Built for the agents platform with a focus on Data.

## Tech Stack
- **Platform:** Agents
- **Language:** TypeScript / JavaScript
- **Framework:** Modern standards for agents
- **Styling:** Vanilla CSS (Premium Aesthetics)
- **Deployment:** Vercel / GitHub Actions / Netlify

- [ ] Performance optimized (fast load times)
