# Test Writer Agent

## Overview
Agent that analyzes source code and generates unit tests with meaningful assertions and edge cases.

## Tech Stack
- **Stack:** Node.js, TypeScript
- **Build:** No build step — open `index.html` in browser

## Features
- Source file analysis
- Test file generation (Jest/Vitest)
- Meaningful assertion generation
- Edge case identification
- Mock/stub suggestions
- Test naming conventions
- Coverage gap detection
- Configurable test framework
- Dry-run preview
- Batch processing

## File Structure
- `index.html — main page`
- `styles.css — styles`
- `app.js — application logic`
- `README.md — documentation`

## Design Guidelines
- **Theme:** Dark mode with Cyber Teal palette
- **Primary:** `#14B8A6`
- **Secondary:** `#2DD4BF`
