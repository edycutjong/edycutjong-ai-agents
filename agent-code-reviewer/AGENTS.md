# Code Reviewer Agent

## Overview
AI agent that reviews PRs for bugs, security issues, and style violations.

## Tech Stack
- TypeScript, Node.js

## Commands
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Features
- PR diff analysis
- Bug pattern detection
- Security vulnerability scanning
- Style violation flagging
- Inline comment suggestions
- Severity scoring
- Summary report generation
- Configurable rule sets
- Language-agnostic analysis
- CI/CD integration

## Design Notes
- Premium, modern CLI output
- No external API keys required for core functionality

## Testing
- Unit tests for core logic
- Target ≥80% code coverage
