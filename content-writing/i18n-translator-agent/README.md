# I18N Translator Agent

Translate i18n JSON files preserving context.

## Features
- JSON input
- Target language
- Context preservation
- Placeholder safety
- Translate multiple files at once
- Quality review
- Diff preview
- Export results in multiple formats

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
