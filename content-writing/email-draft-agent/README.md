# Email Draft Agent

Draft professional email replies from bullet points.

## Features
- Bullet point input
- Tone selector
- Length control
- Multiple drafts
- Thread context
- Auto-append email signature
- Copy result to clipboard
- Built-in reusable templates

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
