# SEO Auditor Agent

## Overview
An AI agent that crawls websites and generates comprehensive SEO reports covering meta tags, headings, links, images, and structured data.

## Tech Stack
- **Stack:** Python, click, beautifulsoup4, requests

## Features
- Crawl pages and extract metadata
- Meta tag completeness check
- Heading hierarchy validation
- Broken link detection
- Image alt text audit
- Structured data validation
- Sitemap.xml analysis
- robots.txt review
- Page speed insights
- Exportable SEO report

## Commands
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Design Guidelines
- **Theme:** Dark mode
- **Primary:** `#FF6B6B`
- **Accent:** `#4ECDC4`
- **Background:** `#1A1A2E`

## Requirements
- No external API keys required for core features
- Works standalone and self-contained
