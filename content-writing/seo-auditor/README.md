# Seo Auditor

An AI agent that crawls websites and generates comprehensive SEO reports covering meta tags, headings, links, images, and structured data.

## Features
- Crawl pages and extract metadata
- Meta tag completeness check
- Heading hierarchy validation
- Broken link detection
- Image alt text audit
- Structured data validation
- Sitemap.xml analysis
- robots.txt review
- Page speed insights
- Exportable SEO report

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
