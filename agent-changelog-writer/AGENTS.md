# Changelog Writer Agent

## Overview
AI agent generating CHANGELOG.md from conventional commits.

## Tech Stack
- TypeScript, Node.js

## Commands
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Features
- Conventional commit parsing
- Semantic version detection
- Grouped by change type
- Breaking change highlighting
- PR/issue link extraction
- Markdown formatting
- Multiple output formats
- Date range filtering
- Monorepo support
- Customizable templates

## Design Notes
- Premium, modern CLI output
- No external API keys required for core functionality

## Testing
- Unit tests for core logic
- Target ≥80% code coverage
