# Docs Qa Agent

Answer questions about project documentation.

## Features
- Doc ingestion
- Question input
- Source citation
- Context retrieval
- Generate follow-up questions and tasks
- Support multiple document sources
- Export results in multiple formats
- Accuracy scoring

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
