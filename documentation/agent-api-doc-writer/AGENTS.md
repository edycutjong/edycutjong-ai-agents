# API Documentation Writer Agent

## Overview
AI agent that generates OpenAPI/Swagger documentation from source code and route definitions

## Tech Stack
- **Stack:** Python, LangChain
- **Build:** `pip install -r requirements.txt` then `python main.py`

## Features
- Parse route definitions
- Generate OpenAPI 3.0 spec
- Request/response schema inference
- Example value generation
- Markdown documentation output
- Multiple language support
- Update existing docs incrementally
- Validate generated spec

## File Structure
- `main.py — agent entry`
- `agent/ — agent modules`
- `requirements.txt — dependencies`
- `README.md — documentation`

## Design Guidelines
- **Theme:** Dark mode with Sunset Orange palette
- **Primary:** `#E17055` | **Secondary:** `#FAB1A0`
- **Accent:** `#00B894` | **Background:** `#1A100A`
- **Border Radius:** 12px | **Font:** Inter or system font stack
- **Animations:** Smooth transitions (200-300ms ease)
- **Style:** Modern, premium, glassmorphism accents

## Requirements
- Fully functional — no placeholder content
- Responsive design | Accessible | No external API keys
