# API Monitor Agent

## Overview
An AI agent that continuously monitors API endpoints for uptime, response time degradation, schema changes, and contract violations.

## Tech Stack
- **Stack:** Python, click, requests

## Features
- Periodic endpoint health checks
- Response time tracking
- Schema drift detection
- Status code monitoring
- Alert threshold configuration
- Historical uptime reports
- Response body diff detection
- Certificate expiry warnings
- Rate limit tracking
- Incident timeline generation

## Commands
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Design Guidelines
- **Theme:** Dark mode
- **Primary:** `#FF6B6B`
- **Accent:** `#4ECDC4`
- **Background:** `#1A1A2E`

## Requirements
- No external API keys required for core features
- Works standalone and self-contained
