# Uuid Generator

UUID Generator — Generate and validate UUIDs (v1, v4, v5).

## Features
- Generate UUID v1, v4, and v5
- Validate UUID format
- Batch generation support
- Namespace-based UUID v5
- Copy-ready output

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
