# Documentation QA Agent

## Overview
AI agent answering codebase questions by reading source files.

## Tech Stack
- TypeScript, Node.js

## Commands
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Features
- Source code indexing
- Natural language queries
- Code reference citations
- Function signature lookup
- Dependency graph navigation
- README parsing
- Comment extraction
- Example code generation
- Caching for performance
- Multi-language support

## Design Notes
- Premium, modern CLI output
- No external API keys required for core functionality

## Testing
- Unit tests for core logic
- Target ≥80% code coverage
