## 2026-03-19 - Streamlit collapsed label accessibility
**Learning:** In Streamlit, when setting `label_visibility='collapsed'`, the first argument (the label string) is still used as the `aria-label` for screen readers. Using a dummy value like 'hidden_label' causes accessibility issues.
**Action:** Always provide a descriptive, localized label rather than a dummy value like 'hidden_label' to ensure accessibility, even if the label is visually hidden.
