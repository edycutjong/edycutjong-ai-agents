## 2026-03-17 - Streamlit Collapsed Labels Accessibility\n**Learning:** In Streamlit, when setting `label_visibility='collapsed'`, the first argument (the label string) is still used as the `aria-label` for screen readers. Using dummy values like 'hidden_label' breaks accessibility.\n**Action:** Always provide a descriptive, localized label as the first argument to Streamlit input components, even when `label_visibility='collapsed'`.
