# AGENTS.md — Json Yaml Converter

## Overview
CLI for JSON/YAML/TOML Converter.

## Tech
- Python 3.10+
- python-dotenv
- pytest

## Features
- Cmd Convert
- Cmd Validate
- Cmd Format
- Cmd Detect

## File Structure
- `README.md`: Readme.Md
- `agent/`: Module directory
- `config.py`: Configuration
- `main.py`: Entry point
- `requirements.txt`: Dependencies
- `tests/`: Module directory

## Commands
- `pip install -r requirements.txt` — Install deps
- `python main.py` — Run agent
- `pytest tests/` — Run tests
