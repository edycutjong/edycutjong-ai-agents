# Performance Profiler Agent

## Overview
An AI agent that profiles web applications, identifies performance bottlenecks, and suggests optimizations for Core Web Vitals.

## Tech Stack
- **Stack:** Python, click, requests

## Features
- Lighthouse audit automation
- Core Web Vitals analysis
- Bundle size recommendations
- Image optimization detection
- Render-blocking resource finder
- Cache strategy suggestions
- Third-party script impact
- Performance budget tracking
- Before/after comparison
- Markdown report generation

## Commands
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Design Guidelines
- **Theme:** Dark mode
- **Primary:** `#FF6B6B`
- **Accent:** `#4ECDC4`
- **Background:** `#1A1A2E`

## Requirements
- No external API keys required for core features
- Works standalone and self-contained
