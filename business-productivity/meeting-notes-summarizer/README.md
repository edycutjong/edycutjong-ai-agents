# Meeting Notes Summarizer

Summarize meeting transcripts into action items.

## Features
- Transcript input
- Key points
- Action items
- Decision log
- Track and attribute contributions
- Summary export
- Follow-up tasks
- Extract and display metadata

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
