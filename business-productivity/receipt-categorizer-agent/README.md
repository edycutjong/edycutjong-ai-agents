# Receipt Categorizer Agent

OCR receipts and auto-categorize expenses.

## Features
- Image upload
- OCR extraction
- Auto-categorize
- Amount detection
- Vendor recognition
- Date parsing
- Export CSV
- Monthly summary

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
