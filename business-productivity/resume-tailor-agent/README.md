# Resume Tailor Agent

Tailor resume to match job descriptions.

## Features
- Resume upload
- JD input
- Keyword matching
- Skills alignment
- Experience rewrite
- ATS optimization
- Calculate and display quality score
- Export results in multiple formats

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
