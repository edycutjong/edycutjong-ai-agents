# Ip Lookup

IP Lookup — Look up IP address geolocation and network information.

## Features
- Geolocation lookup (city, country)
- ISP and organization info
- Reverse DNS lookup
- IPv4 and IPv6 support
- Batch IP lookups

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
