# File Hash Generator

File Hash Generator — Generate file hashes (MD5, SHA-256, SHA-512) for integrity checks.

## Features
- Generate MD5, SHA-1, SHA-256 hashes
- SHA-512 support
- Hash file contents or strings
- Compare hash values
- Batch file hashing

## Usage
```bash
python main.py
```
