# Color Converter

Color Converter — Convert between color formats (HEX, RGB, HSL).

## Features
- Convert HEX to RGB and HSL
- Convert RGB to HEX and HSL
- Convert HSL to HEX and RGB
- Color name lookup
- Batch color conversion

## Usage
```bash
python main.py
```

## Testing
```bash
pytest tests/ -v --cov=. --cov-report=term-missing
```
