# Test Writer Agent

## Overview
AI agent generating unit tests for uncovered functions.

## Tech Stack
- TypeScript, Node.js

## Commands
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm test`

## Features
- Code coverage analysis
- Test case generation
- Edge case identification
- Mock/stub creation
- Assertion generation
- Test framework detection
- File placement logic
- Import resolution
- Parameterized tests
- Coverage report comparison

## Design Notes
- Premium, modern CLI output
- No external API keys required for core functionality

## Testing
- Unit tests for core logic
- Target ≥80% code coverage
